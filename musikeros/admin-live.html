Panel Admin — Live + mensaje especial

Desde aquí puedes cambiar la plataforma del live, pegar la URL actual y mostrar un mensaje especial apenas entren a la web.

YouTube: watch, youtu.be, live, embed o ID directo.
Facebook: pega el enlace del video o live público.

Si dejas la URL vacía, el live queda apagado.

Sugerencia: usa un título corto y un mensaje breve. Si cierras el live con “Apagar live”, también se limpiará el aviso.
